# Balaji Fast Food Sales Dashboard

## 📊 Project Overview
This project focuses on analyzing fast food sales data for Balaji using Power BI. The dashboard provides insights into sales performance, customer trends, and product demand to support better business decisions.

## 🔧 Tools Used
- Power BI
- Excel / Data Source

## 📈 Key Insights
- Identified top-performing food items contributing to revenue
- Analyzed daily and monthly sales trends
- Compared category-wise sales performance
- Observed peak sales periods and customer buying patterns
- Highlighted low-performing items for improvement
